Lisp Implementations

Common Lisp Implementations

  • CLiki: Comparison of actively developed Common Lisp implementations (this is probably not longer up to date).
  • Armed Bear Common Lisp (ABCL): a full implementation of the Common Lisp language featuring both an interpreter and a compiler, running in the JVM. ABCL is covered by the GNU General Public License with Classpath exception.
    See also: Erik Huelsmann: Development and Deployment of Common Lisp Applications on the JVM (YouTube, 17.11.2013, 29 minutes): video of a presentation at ECLM 2013.
  • Franz: Common Lisp Tools: proprietary enterprise-level tools (Allegro CL® with AllegroCache®).
  • Christian Schafmeister: Clasp: a open source Common Lisp environment. See the licences (LGPL 2.1, Memory Pool System Kit Open Source License, etcetera).
    See also: Christian Schafmeister: Clasp: Common Lisp using LLVM and C++ for Molecular Metaprogramming (YouTube, 15.06.2015, 58 minutes): a Google TechTalk held on 10.06.2015.
  • Clozure Common Lisp (CL): an open source Common Lisp implementation (licence: Apache 2.0). The source code is available on GitHub.
  • CLISP: an open source portable ANSI Common Lisp implementation (licence: GPL 2.0).
  • CMUCL: a free implementation of the Common Lisp programming language which runs on most major Unix platforms (this includes GNU Linux, BSD and Mac OS X). Most of the source code is in the public domain.
  • Corman Technologies: Corman Common Lisp: a commercial Common Lisp implemenation for Microsoft Windows.
  • Embeddable Common-Lisp (ECL. An open source implementation whose source code is available on GitLab under the terms of the LGPL licence.
  • GCL: GNU Common Lisp.
  • Hedgehog: a very concise implementation of a LISP-like language for low-end and embedded devices. The source code is available on GitHub under the conditions of the BSD-3-Clause licence but has not been updated since 2014.
  • LispWorks: a proprietary Common Lisp implementation.
  • ManKai Common Lisp (MKCL): aims to be a full implementation of the Common Lisp language in compliance with the ANSI X3J13 Common Lisp standard. Licence: LGPL, with a few minor exceptions.
  • mocl: a proprietary Lisp implementation for mobile platforms (iOS and Android) and Mac OS X. It runs an expansive subset of Common Lisp.
  • University of Birmingham School of Computer Science: Poplog: a free implementation of Lisp, Prolog, Standard ML and POP-11. The programming environment was originally created for teaching and research in artificial intelligence. See also Poplog in Wikipedia.
  • Steel Bank Common Lisp (SBCL): an open source Common Lisp implementation (licence: MIT). There is a mirror of the source code on GitHub. According to the article Steel Bank Common Lisp (SBCL) in Wikipedia, it features a high performance native compiler, Unicode support and threading.
  • Scieneer Common Lisp: proprietary Common Lisp implementation.

Implementations of Other Lisp Variants

  • OpenLisp by Eligis: A full and efficient ISO/IEC 13816:2007(E) ISLISP Implementation. OpenLisp is a commercial product. OpenLips ports can be downloaded free of charge for an exclusive non-commercial usage.
  • youtoo: a public domain implementation of the object-oriented dynamic programming language EuLisp.

Editors and IDEs

  • SLIME: The Superior Lisp Interaction Mode for Emacs. The source code is available on GitHub as public domain software.
  • Climacs: a free Emacs-like editor in Common Lisp.
  • Portable Hemlock: a free Emacs-like editor.
  • Hemlock: an extensible Emacs-like text editor embedded in CMUCL.
  • Lispbox: an IDE that is essentially a pre-configured packaging of the Emacs editing environment, SLIME (The Superior Lisp Interaction Mode for Emacs), the Quicklisp library manager, and the Clozure Common Lisp compiler. The package was last updated in 2011. The source code is available on GitHub. (Licence: BSD.)
  • Franz: Allegro CL: commercial IDE for Lisp.
  • Franz: Allegro CL Free Express Edition Downloads: downloading this version requires that you agree to the Allegro CL Free Express Edition license agreement.

Lisp Libraries and Tools

  • Edi Weitz: Common Lisp Libraries by Edi Weitz (on GitHub, unfortunately mostly without a licence file, so check the licence headers in the source code).
  • Marijn Haverbeke: Postmodern: a Common Lisp library for interacting with PostgreSQL databases. The source code is available on GitHub under a zlib-style licence.
  • McCLIM: an open source GUI toolkit that implements the Common Lisp Interface Manager CLIM II Specification. The source code is hosted on GitHub and licensed under the GNU Library General Public License. For a list of applications using CLIM, see the Wikipedia article Common Lisp Interface Manager.
  • Peter Herth: LTK - The Lisp Toolkit (the code has not been updated since 2006 and there is no licence).
  • CLX: an open source X11 client library for Common Lisp.
  • Nathan Froyd: Ironclad: a cryptography library written entirely in Common Lisp. Its source code is available on GitHub under the terms for the BSD-3-Clause licence.
  • lisp-unit: A Test Framework for Common Lisp in the style of JUnit, designed and implemented with simplicity of use in mind. The code was last updated in 2016 and has no licence file.
  • ASDF: the de facto standard build facility for Common Lisp.
  • Quicklist beta: a library manager for Common Lisp.
  • Caveman 2: The real web framework. The source code is available on GitHub under the terms of the LLGPL licence.
  • Weblocks: a web framework written in Common Lisp. Its source code is available on GitHub (licence: LGPL 2.1).
  • UnCommon Web: a web application framework for Common Lisp.
  • Marc Kuo: Open-VRP: an open source framework for modelling the vehicle routing problem available under the terms of the Lisp Lesser GNU Public License (LLGPL). The project is briefly mentioned in the blogpost Why I love Common Lisp and hate Java (27.01.2012).
  • Wolfgang Goerigk, Ulrich Hoffmann, Heinz Knutzen (Christian-Albrechts-Universität zu Kiel): CLiCC: The Common Lisp to C Compiler: developed in 1991 – 1996 and probably only of historical interest. Available under the GNU General Public License version 2.0.
