Chess

Playing Chess Online / Internet Chess Servers

  • Lichess.
  • Chess.com: internet chess server that includes a forum.
  • Free Internet Chess Server (FICS): an internet chess server run by volunteers.
  • Internet Chess Club (ICC): an internet chess server that partnered with the United States Chess Federation (USCF) so it can organise USCF-rated tournaments.
  • FIDE Online Arena: internet chess server run by the Fédération internationale des échecs (FIDE).
  • ChessCube.
  • chess24.
  • Glaurung CHESS.
  • playchess.com: internet chess server run by ChessBase.
  • Europe Echecs: Jouez aux échecs en ligne.
  • SchemingMind.com: a correspondence chess server.
  • Zeit Online: Schach.

Chess Software

Integrated Software and Chess Engines

  • Stockfish (on GitHub): a free and open source chess engine for GNU Linux, Microsoft Windows and Mac OSX. See also Stockfish Chess.
  • Komodo: a commercial chess engine developed by developed by Don Dailey, Mark Lefler, and supported by Grandmaster Larry Kaufman.
  • Houdini Chess Engine: commercial chess engine eveloped by Belgian programmer Robert Houdart.
  • Shredder: a commercial chess software that has frequently won the World Computer Chess Champion.
  • Rybka: a commercial chess engine developed by International Master Vasik Rajlich.
  • Crafty: a free and open source chess engine developed by Dr. Robert M. (Bob) Hyatt.
  • Spike: a chess engine that supports both the UCI and Winboard protocols.
  • HIARCS Chess Software.
  • GNU Project: GNU Chess.
  • Tim Mann: GNU Chess.
  • ChessGenius: chess programs developed by Richard Lang.
  • OpenChess for PalmOS. See also the public-facing site.
  • jChecs: open source chess software in Java, designed to introduce the basics of computer chess programming concepts.
  • Winboard 4.5 Accessible Chess: Chess for the Blind for the JAWS or NVDA Screen Readers (open source).
  • Java Chess.
  • Mobile Chess and Flash Chess: open source chess software. Mobile Chess is written in Java ME.
  • Brutal Chess: open source chess software inspired by Battle Chess from the 1980s. The code was last updated in 2013.
  • Fruit: chess engine developed by Fabien Letouzey which is no longer being developed. The source code of version 2.1 is still available.
  • Mediocre Chess: open source chess engine.
  • Carballo Chess Engine: open source chess engine.
  • Pharaon: chess engine developed by Franck Zibi (and previously known as ZChess). There have been no updates since 2006.
  • Monarch: chess engine developed by Steve Maughan, which had around 2100 ELO when development was discontinued in 2006.

Chess Interfaces

  • Arena: chess GUI for GNU Linux and Microsoft Windows. It supports both the UCI and Winboard protocols. The website also has an Engine Links Collection.
  • GNOME Chess: open source chess interface written in Vala. See also the GNOME Chess Manual.
  • PyChess: open source chess interface written in Python that can connect to the Free Internet Chess Server (FICS) or be used as an interface for many chess engines. Its built-in chess engine, which is also written in Python, is rather weak.

Chess Database Software

  • ChessX: Chess Database and PGN viewer.
  • Scid: open source chess database application for GNU Linux and Microsoft Windows.
  • ChessShell for PC/Mac/Linux: Chess Database application.
  • hessDB - a Free Chess Database: open source chess database for Microsoft Windows, GNU Linux and Mac OS. Last updated in 2013.
