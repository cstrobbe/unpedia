Learning Rust

  • Rust by Example: a collection of runnable examples that illustrate various Rust concepts and standard libraries. The source code is available on GitHub (licence: Apache 2.0).
  • Rust in 7 Programs: This tutorial is designed to expand upon and build off of the tutorial offered at rust-lang.org, going in greater depth and (optimally) providing a better explanation of certain topics.
  • Rust for Rubyists: archived code for a tutorial originally hosted at RustForRubyists.org. The source code has not been updated since 2015 but is still available under the terms of the AGPL 3.0 licence.
  • Klock, Felix II: Rust: Safe Systems Programming with the Fun of FP, Code Mesh 2013.
    The video is also available elsewhere: Rust: Safe Systems Programming with the Fun of FP on Vimeo, Rust: Safe Systems Programming with the Fun of FP on YouTube (47 minutes).

Editors and IDEs

  • Rust and IDEs, Rust Forge (no date).
  • racer: an open source Rust code completion utility that can be integrated into editors and IDEs such as Eclipse, Emacs, Sublime Text, Atom, Vim and Kate. (Licence: MIT.)
  • Are we (I)DE yet?: a webpage that tracks the status of Rust support in editors and IDEs.
  • PistonDevelopers: VisualRust: an open source Visual Studio extension for Rust (licence: MIT).
  • SolidOAK: a simple open source IDE for Rust. According to the website, It starts off in "Easy Mode" (locked in insert mode) for Vim newbies, but can be toggled off. The source code is available on GitHub (licence: Unlicense).
  • Ride: an open source Rust IDE. In early 2018, its developers consider it not yet in a usable state. (Licence: MIT.)
  • Zeus IDE: Rust Zeus Macros.

Rust Libraries and Tools

  • Awesome Rust: A curated list of Rust libraries and resources, maintained as a community effort.
  • Vladimir Matveev: xml-rs: an open source XML library that is heavily inspired by Java Streaming API for XML (StAX) (licence: MIT).
  • PistonDevelopers: piston: A modular game engine written in Rust (licence: MIT).
  • PistonDevelopers: Conrod: An easy-to-use, 2D GUI library written entirely in Rust (licence: MIT or Apache 2.0).
  • Sean Marshallsay: rusty-cheddar: A Rust crate for automatically generating C header files from Rust source file. (Source code available without a licence.)
  • Rusticata: Rust parsers for Suricata. Licence: LGPL 2.1.
