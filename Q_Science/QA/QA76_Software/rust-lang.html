Books about Rust

  • Klabnik, Steve; Nichols, Carol: The Rust Programming Language. No Starch Press, May 2018. (584 pages) ISBN 978-1-59327-828-1.
    From the publisher's book description:
    The Rust Programming Language is the official book on Rust, an open-source, community-developed systems programming language that runs blazingly fast, prevents segfaults, and guarantees thread safety. This is the undisputed go-to guide to Rust, written by two members of the Rust core team, with feedback and contributions from 42 members of the community. The book assumes that you’ve written code in another programming language but makes no assumptions about which one, meaning the material is accessible and useful to developers from a wide variety of programming backgrounds.

    See also The Rust Programming Language (online version). The source code used in the book can be found in the rust-lang/book repository on GitHub.
    An edition updated to cover Rust 2018 was published in August 2019 (ISBN 978 1 7185 0044 0).
  • Blandy, Jim; Orendorff, Jason: Programming Rust: Fast, Safe Systems Development. O'Reilly, November 2017. (622 pages; example code on GitHub)
  • Balbaert, Ivo: Rust Essentials. Second edition. Packt, 2017. (264 pages) ISBN 9781788390019.
  • Milanesi, Carlo: Beginning Rust: From Novice to Professional. Apress, April 2018. ISBN 978-1-4842-3467-9.
    This book is mainly for programmers with some experience in C and/or C++.

Editors and IDEs

  • Rust and IDEs, Rust Forge (no date).
  • racer: an open source Rust code completion utility that can be integrated into editors and IDEs such as Eclipse, Emacs, Sublime Text, Atom, Vim and Kate. (Licence: MIT.)
  • Are we (I)DE yet?: a webpage that tracks the status of Rust support in editors and IDEs.
  • PistonDevelopers: VisualRust: an open source Visual Studio extension for Rust (licence: MIT).
  • SolidOAK: a simple open source IDE for Rust. According to the website, It starts off in "Easy Mode" (locked in insert mode) for Vim newbies, but can be toggled off. The source code is available on GitHub (licence: Unlicense).
  • Ride: an open source Rust IDE. In early 2018, its developers consider it not yet in a usable state. (Licence: MIT.)
  • Wilfred Hughes: Rust Emacs: a community-driven port of Emacs to Rust (GPL 3.0).
  • Zeus IDE: Rust Zeus Macros.

Rust Libraries and Tools

  • Awesome Rust: A curated list of Rust libraries and resources, maintained as a community effort.
  • Jonathan Reem: stainless: a testing framework available under the terms of the MIT framework.
    See also the Testing chapter in the Rust documentation.
  • Vladimir Matveev: xml-rs: an open source XML library that is heavily inspired by Java Streaming API for XML (StAX) (licence: MIT).
  • PistonDevelopers: piston: A modular game engine written in Rust (licence: MIT).
  • PistonDevelopers: Conrod: An easy-to-use, 2D GUI library written entirely in Rust (licence: MIT or Apache 2.0). See also the Piston website.
  • Sean Marshallsay: rusty-cheddar: A Rust crate for automatically generating C header files from Rust source file. (Source code available without a licence.)
  • Rusticata: Rust parsers for Suricata. Licence: LGPL 2.1.
  • Exonum: a framework for blockchain projects. See Exonum's GitHub repositories. The code in Exonum's core repository is available under the terms of the Apache 2.0 licence.
  • Rustls: a TLS library available under three licences: Apache 2.0, MIT and ISC.
  • XML parsers:
    • Rahm, Eric: A Rust-based XML parser for Firefox, Eric Rahm's blog, 24.05.2017.
    • quick-xml: XML pull parser and writer available under the MIT licence.
    • xml-rs: XML library inspired by the Streaming API for XML (StAX), available under the MIT licence.
  • rust-encoding: character encoding support for Rust (licence: MIT).
  • validator: Simple validation for Rust structs (licence: MIT).
  • Tock: an embedded operating system whose kernel and device drivers are written in Rust. See also the Tock Embedded OS GitHub repositories.
  • rust-cpython: Rust bindings for the python interpreter (MIT licence).
