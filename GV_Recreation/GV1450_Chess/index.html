Chess

Playing Chess Online / Internet Chess Servers

  • Lichess.
  • Chess.com: internet chess server that includes a forum.
  • Free Internet Chess Server (FICS): an internet chess server run by volunteers.
  • Internet Chess Club (ICC): an internet chess server that partnered with the United States Chess Federation (USCF) so it can organise USCF-rated tournaments.
  • FIDE Online Arena: internet chess server run by the Fédération internationale des échecs (FIDE).
  • ChessCube.
  • chess24.
  • Glaurung CHESS.
  • playchess.com: internet chess server run by ChessBase.
  • Europe Echecs: Jouez aux échecs en ligne.
  • SchemingMind.com: a correspondence chess server.
  • Zeit Online: Schach.

Chess Software

Integrated Software and Chess Engines

  • Stockfish (on GitHub): a free and open source chess engine for GNU Linux, Microsoft Windows and Mac OSX. See also Stockfish Chess. (See also Marco Costalba's Stockfish fork, which has not been updated since 2014.)
  • Komodo: a commercial chess engine developed by developed by Don Dailey, Mark Lefler, and supported by Grandmaster Larry Kaufman.
  • Houdini Chess Engine: commercial chess engine eveloped by Belgian programmer Robert Houdart.
  • Shredder: a commercial chess software that has frequently won the World Computer Chess Champion.
  • Rybka: a commercial chess engine developed by International Master Vasik Rajlich.
  • Crafty: a free and open source chess engine developed by Dr. Robert M. (Bob) Hyatt.
  • Spike: a chess engine that supports both the UCI and Winboard protocols.
  • HIARCS Chess Software.
  • GNU Project: GNU Chess.
  • Alberto Alonso Ruibal: Carballo Chess Engine: a chess engine written in Java and available on GitHub under the GPL 3.0 licence.
  • Alberto Alonso Ruibal: Karballo Chess Engine: a Kotlin of Carballo. The engine is available on GitHub under the MIT licence.
  • andermvdb: chess22k: a chess engine that uses the UCI protocol. According to its developer, the engine scores around 2900 Elo. chess22k is written in Java and available on GitHub under the GPL 3.0 licence.
  • Phokham Nonava: flu: an open source chess engine in Java, avaiable under the LGPL 3.0 licence. Flux supports the UCI protocol.
  • Phokham Nonava: pulse: a didactice chess engine written in a combination of Java and C++. The source code is available under the MIT licence.
  • Tim Mann: GNU Chess.
  • ChessGenius: chess programs developed by Richard Lang.
  • OpenChess for PalmOS. See also the public-facing site.
  • jChecs: open source chess software in Java, designed to introduce the basics of computer chess programming concepts.
  • Winboard 4.5 Accessible Chess: Chess for the Blind for the JAWS or NVDA Screen Readers (open source).
  • Java Chess.
  • Mobile Chess and Flash Chess: open source chess software. Mobile Chess is written in Java ME.
  • Pawel Koziol: Rodent_II: an open source chess engine written in C++ and C, and available under the GPL 2.0 licence.
  • Romain Goussault: Deepov: an open source chess engine written in C++ and available under the GPL 2.0 licence.
  • Niklas Fiekas: stockfish.js: open source chess engine Stockfish compiled to JavaScript and WebAssembly using Emscripten. This engine is used for analysis on Lichess. The code is available under the GPL 3.0 licence.
  • Niklas Fiekas: python-chess: a chess library written in Python and available under the GPL 3.0 licence. For the latest info about the code, see the python-chess website.
  • ttu-fpclub: Kawa Chess: a chess engine written in Kawa Scheme (a Scheme dialect) that can be run on the Java Virtual Machine using a jar file from the Kawa Scheme project. The source code has no licence.
  • Stephen Fleischman: Pleco an open source chess engine inspired by Stockfish and written in Rust. The engine supports the UCI protocol. The code is available under the MIT licence.
  • Jon Dart: Arasan: an open source multi-platform chess engine written in C++. It supports multi-threading for higher performance. The code is available under the MIT licence.
  • Moh'd Kayali: cicada: an open source chess engine written in Rust that was created as a hobby project. It is compatible with the UCI protocol and ranks at around 1450 - 1650 ELO according to its developer. The code is available under the MIT licence.
  • Dylan Hunn: Dragontooth: a fast, UCI-compliant chess engine written in Go. In early 2018, this project was in beta state. (Licence: GPL 3.0.)
  • Leonid Rozenblyum: chess: an open source chess engine in Java that is still under development (not yet ready for use in February 2018). The code is available under the Apache 2.0 licence.
  • Fenimore: Ghess: an open source chess engine written in Go. In early 2018 it was not complete (e.g. pawns could only promot to queen). The code is available under the GPL 3.0 licence.

Open source projects that involve machine learning or other types of artificial intelligence:

  • Truls Edvard Stokke: Zerofish: an open source project in Python that aims to develop an implementation of the AlphaZero algorithm for chess. The code is available under the MIT licence.
  • Ashudeep Singh: ConvChess: Convolutional Neural Networks learn chess moves. (MIT licence.)

The following chess engines are (probably) no longer being developed:

  • Pawel Koziol: cpw-engine: an open source didactic chess engine for the Chess Programming Wiki. The code is in C++ and C and has no licence.
  • Matthew Voss: Chess: an open source chess engine written in Java. The code has not been updated since 2016 and has no licence. The developer points out: I wrote this before I had any experience with enterprise software design. It was an academic exercise and turned out to be a clever chess player, but I would definitely code this much cleaner if I were to redo it.
  • Romain Goussault: Jeepov: an open source chess engine written in Java and Groovy. The code has not been updated since 2014 and has no licence.
  • Sloppy: a chess engine written in C that is available under GPL 3.0 licence. The code has not been updated since 2009. For details, see the Sloppy website.
  • Mark Qian: Chess-engine-in-Lisp: a chess engine written in Common Lisp. The code has not been updated since 2015 and has no licence.
  • Mike Vollmer: lalla-chess: an open source chess engine written in Common Lisp. The code is available under the BSD-2-Clause licence and has not been updated since 2013.
  • Nick Meyer: crust: an open source chess engine written in Rust. The code is available under the MIT licence and has not been updated since 2015.
  • Marat Tukhvatullin: chess4j: an open source chess engine in Java. The code has not been updated since 2014 and has no licence.
  • Mahoneko / Ouzned: JavaChess: a small portable chess engine in Java. The code has no licence.
  • Mahoneko / Ouzned: HaskellChess: a small chess engine in Haskell, written in order to become familiar with the Haskell programming language. The code has not been updated since 2012 and has no licence.
  • Brutal Chess: open source chess software inspired by Battle Chess from the 1980s. The code was last updated in 2013.
  • Fruit: chess engine developed by Fabien Letouzey which is no longer being developed. The source code of version 2.1 is still available.
  • Mediocre Chess: open source chess engine.
  • Carballo Chess Engine: open source chess engine.
  • Christopher Wellons: October Chess Engine: a simple open source chess engine written in Java that can be run stand-alone or as an applet. The code has not been updated since 2012 and is available under the “Unlicense”.
  • Pharaon: chess engine developed by Franck Zibi (and previously known as ZChess). There have been no updates since 2006.
  • Monarch: chess engine developed by Steve Maughan, which had around 2100 ELO when development was discontinued in 2006.

Chess Interfaces

  • Arena: chess GUI for GNU Linux and Microsoft Windows. It supports both the UCI and Winboard protocols. The website also has an Engine Links Collection.
  • GNOME Chess: an open source chess interface written in Vala. See also the GNOME Chess Manual.
  • Cute Chess is an open source chess interface by Ilari Pihlajisto, Arto Jonsson and contributors. The program is written in C++ and the Qt framework and is available under the MIT licence.
  • PyChess: an open source chess interface written in Python that can connect to the Free Internet Chess Server (FICS) or be used as an interface for many chess engines. Its built-in chess engine, which is also written in Python, is rather weak.

Chess Database Software

  • ChessX: Chess Database and PGN viewer.
  • Scid: open source chess database application for GNU Linux and Microsoft Windows.
  • ChessShell for PC/Mac/Linux: Chess Database application.
  • hessDB - a Free Chess Database: open source chess database for Microsoft Windows, GNU Linux and Mac OS. Last updated in 2013.

Other Chess Software

  • blitz-tactics is an open source chess tactics trainer written in Ruby and JavaScript. The code is used on the site BlitzTactics.com, where tactics puzzles are group in levels or sets of 15–20 tactics. The source code is available under the MIT licence.
  • Pachyderm: Chess: an open source MapReduce job to explore blunders in chess games. The code is written in Python and Go but has no licence.
  • chessboard.js is a JavaScript chessboard component for integration in web pages. It is available under the MIT licence. chessboard.js can be combined with chess.js, a JavaScript library for move generation and validation, piece movement and other chess-related functions. It does not contain an AI component that plays chess. chess.js is available under the BSD-2-Clause licence.
  • Thibault Duplessis / ornicar: lila: the source code of the chess server lichess.org, written Scala, Typescript, HTML, JavaScript and a few other languages. The code is available under the GNU Affero General Public License.
  • Thibault Duplessis / ornicar: scalachess: a chess API written in Scala and available under the MIT licence.
  • Thibault Duplessis / ornicar: chessground: a Mobile/Web chess UI for lichess.org. The source code is available under the GPL 3.0 licence.
  • Thibault Duplessis / ornicar: lichess-db: code to export lichess game database in a standard PGN format. The source code is written mostly in Scala and is available under the AGPL 3.0 licence.
  • Niklas Fiekas: syzygy-tables.info: User interface and public API for probing Syzygy endgame tablebases. See the website Syzygy endgame tablebases. The source code is available under the AGPL 3.0 licence.
  • Marco Costalba: chess_db: a small program in C++ with some Python for indexing PGN files. The code is available under the GPL 3.0 licence.
  • jaxonspencer: chess_fen.py: an FEN plugin for the spaced-repetition program Anki. The code is written in Python, has no licence and has not been updated since 2013.
  • Drew Hannay: chess: an open source Java application for defining and playing chess variants. The developer invites contributions but has not specified a licence for the source code. See also the website Chess Crafter.
  • Dylan Hunn: Dragontooth Movegen: a fast, no-compromises chess move generator written entirely in Go. (Licence: GPL 3.0.)
