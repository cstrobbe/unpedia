JavaScript Programming Language

JavaScript Documentation / JavaScript in General

  • JavaScript reference, MDN.
  • ECMAScript® 2015 Language Specification, ECMA International. This is the official specification of EcmaScript version 6.
  • ECMAScript® 2017 Language Specification ECMA International. This is the official specification of EcmaScript version 8.
    The most recent drafts are available on ECMA's GitHub repository.
  • Copes, Flavio: The Complete JavaScript Handbook, freeCodeCamp.org, 30.10.2018 (91-minute read). The JavaScript Handbook follows the 80/20 rule: learn 80% of JavaScript in 20% of the time. You can also download the document in other formats: PDF, ePub or Mobi.

Learning JavaScript

  • Haverbeke, Marijn: Eloquent JavaScript: online book. The printed version is now in its third edition.
  • Rauschmayer, Axle: Speaking JavaScript: An In-Depth Guide for Programmers. O'Reilly, 2014.
    The book's website has a link to the HTML version of Speaking JavaScript.
  • Osmany, Addy: Learning JavaScript Design Patterns: A JavaScript and jQuery Developer's Guide. O'Reilly Media, 2012.
    See also the Learning JavaScript Design Patterns website, where a more recent version is available under the terms of the Creative Commons Attribution-Noncommercial-No Derivative Works 3.0 unported licence. The book's source code is available on GitHub.
  • You Don't Know JS: a series of books available on GitHub. The series contains the following titles, which can be read online:
    • Up & Going
    • Scope & Closures
    • this & Object Prototypes
    • Types & Grammar
    • Async & Performance
    • ES6 & Beyond
  • Lindley, Cody: JavaScript Enlightenment: From Library User to JavaScript Developer. O'Reilly Media, 2012.
    See also the JavaScript Enlightenment website.
  • Lindley, Cody: DOM Enlightenment: Exploring JavaScript and the Modern DOM. O'Reilly Media, 2013.
    See also the DOM Enlightenment website.

Other books are listed on the website JSbook.

Due to the evolution of the JavaScript language and of JavaScript practices, the following books are mainly of historical interest:

  • Koch, Peter-Paul: ppk on JavaScript. New Riders (Peachpit Press), 2006. (528 pages) ISBN 978-0-321-42330-6.
    See also Koch's webpage about ppk on JavaScript.

Advanced Topics

  • Podjarny, Guy: Writing Secure JavaScript (YouTube, 24 minutes, 06.04.2018): presentation at JS Kongress Munich, November 2016. Podjarny is co-founder and CEO of Snyk. The presentation mainly demonstrates security vulnerabilities in JavaScript libraries and modules.
  • Smith, Jared: JavaScript Security: What You Need to Know to Write Secure Applications in JS (YouTube, 41 minutes, 28.11.2016): presentation at Nodevember 2016.
  • Vishnia, Opher: Wait, you can do that with JavaScript…!? (YouTube, 29 minutes, 09.04.2018): presentation at JSConf Iceland, 2018.
  • Düüna, Karl: Secure Your Node.js Web Application: Keep Attackers Out and Users Happy. The Pragmatic Bookshelf, 2016. (230 pages) ISBN 978-1-68050-085-1.
  • Burnett, Nick; Biernat, Patrick; Gaasedelen, Markus: Weaponization of a JavaScriptCore Vulnerability, Ret2 Systems Blog, 11.07.2018.
