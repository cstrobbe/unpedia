Online Privacy Protection

Browser Extensions and Online Tools

  • Privacy Badger: a browser extension developed by the Electronic Frontier Foundation (EFF).
  • NoScript: browser extension for FireFox and SeaMonkey that disables JavaScript and allows you to selectively enable scripts on a script-by-script and site-by-site basis. This is very effective but requires more work by the user.
  • Tor Project: Tor is free software and an open network that helps you defend against traffic analysis, a form of network surveillance that threatens personal freedom and privacy, confidential business activities and relationships, and state security.
  • Panopticlick: online tool by the EFF that can analyze how well your browser and add-ons protect you against online tracking techniques.
  • BrowserSpy: a site where you can see just how much information your browser reveals about you and your system.
  • IP Check.
  • WebRTC IP Leak VPN / Tor Test, PrivacyTools.io.
  • Off-the-Record Messaging.

Encrypting Hard Drives and External Media

  • VeraCrypt: a free open source disk encryption software for Windows, Mac OSX and Linux.
  • Buttler, Peter: How To Encrypt Your Hard Drive? – Encryption Software 2018, Be Encrypted blog, 01.01.2018.
  • Wallen, Jack: How to encrypt a USB flash drive with VeraCrypt, TechRepublic, 28.12.2017.
    The author demonstrates the process using Elementary OS.
  • Douma, Andrew: Full Disk Encryption with VeraCrypt, Security Streak on Medium, 22.02.2017 (seven-minute read).
  • How to encrypt your hard disk in Windows 10, IT Pro, 05.09.2016.
  • Kishore, Aseem: How to Securely Encrypt a USB Flash Drive, Online Tech Tips, 11.12.2015.
    The author explains how to encrypt a USB flash drive using BitLocker on Microsoft Windows, using VeraCrypt on Linux and using hardware-encrypted drives (e.g. the IronKey range).
  • Schüßler, Jan: USB-Medien sicher verschlüsseln, c't Magazin, 22.06.2018 (behind a paywall).
  • Comparison of disk encryption software (Wikipedia).
  • Schneier, Bruce: TrueCrypt's Deniable File System, Schneier on Security, 18.07.2008.
    Note that the development of TrueCrypt was discontinued in 2014. There are several alternatives, such as VeraCrypt.
  • Barnatt, Christopher: VeraCrypt Encrypted USB Drive (ExplainingComputers on YouTube, 16 minutes, 14.02.2016). This video demonstrates to use VeraCrypt on Windows 10 to create an encrypted file container and then how to copy VeraCrypt and the encrypted file to a USB flash drive. Before moving the file, he reformats the flash drive with the exFAT file system. After this, Barnatt demonstrates how to mount the flash drive and the encrypted volume. Finally, he shows how to mount the flash drive and the encrypted volume on a different (older) Windows PC.
  • Linux GUI encryption (2018) (Average Linux User on YouTube, 10 minutes, 17.05.2018). This video shows how to install VeraCrypt (using the command line), how to create an encrypted container, how to use a keyfile and how to mount the encrypted container (which you can then find in your file manager).
  • AESCrypt: an open-source file encryption program for Windows, Linux, MacOS, Android and iOS.
