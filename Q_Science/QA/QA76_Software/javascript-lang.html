Performance

  • Osmany, Addy: The Cost of JavaScript, Addy Osmany on Medium, 01.08.2018. Blog post about the cost (in the sense of an inferior user experience) of sending heavey JavaScript payloads to the browser. Osmany presents a few solutions, such as code-splitting, regular auditing of JavaScript bundles, monitoring and optimising JavaScript performance, removing unused JavaScript and using a performance budget. Osmany also made a presentation on the cost of JavaScript at the Fluent 2018 conference.
  • Still, Andy: Web Performance Warrior: The Business of Speed. O'Reilly. (Free e-book)
  • Hogan, Lara Callender: Designing for Performance: Weighing Aesthetics and Speed. O'Reilly, 2014. See the website for Designing for Performance.
  • Walton, Philip: Why Web Developers Need to Care about Interactivity Philip Walton, 19.12.2017.
