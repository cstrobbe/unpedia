Learning C

  • Learn C Programming: The Definitive Guide: an online tutorial by Programiz in Nepal.
  • Kernighan, Brian W.; Ritchie: The C Programming Language. Second edition. Prentice Hall, 1988. (274 pages) ISBN 9780131103627. The first edition was published in 1978. The second edition describes C as defined by the ANSI standard (C89). (See also the Wikipedia article.) The book contains exercises; solutions to these exercises are available in several places.
    • K&R solutions in the clc-wiki.
    • Thomas van der Burgt: KnR-The-C-Programming-Language-Solutions (chapters 1 – 8, on GitHub, no licence).
    • Lin Ke-Fong: tcpl (chapters 1 – 8, with compilation examples, on GitHub, no licence).
    • Andrew Laing: KerniganAndRitchieExercises (chapters 1 – 7, on GitHub, no licence).
    • Syam.G.Krishnan: ANSI_C_KandR (chapters 1 – 6, on GitHub, no licence).

Editors, IDEs and Other Tools

  • tis-interpreter is a tool that detects subtle bugs in C programs that may not have eye-visible effects when executing the same programs compiled in the traditional way. The tis-interpreter source code is available on GitHub.
