Python Libraries and Software

  • IPython: an open-source architecture for interactive computing. See also Jupyter and the list of books and videos, which mainly lists resources related to data science and data visualisation.
  • Jupyter: an open-source project that exists to develop open-source software, open-standards, and services for interactive computing across dozens of programming languages.
  • sympy: a open-source computer algebra system written in pure Python.
  • Natural Language Toolkit (NLTK).
  • lxml: an open-source library for parsing XML and HTML that provides bindings for C libraries libxml2 and libxslt. See also lxml on GitHub, where the code is available under the terms of the BSD licence.
  • PDFMiner: PDFMiner is a tool for extracting information from PDF documents. For source code, see PDFMiner on GitHub.
  • PyPDF2: A Pure-Python library built as a PDF toolkit.
  • PDFix SDK: sample code for a Python library for various types of PDF manipulation. Although the code can be inspected on GitHub, it does not have an open-source licence and should therefore not be considered as open-source software. The actual SDK needs to be downloaded from PDFix.net.
