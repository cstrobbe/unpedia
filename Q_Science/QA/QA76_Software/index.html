Becoming a Self-Taught Programmer

  • Atwood, Jeff: Please Don't Learn to Code, Coding Horror, 15.05.2012.
    This blog post comments on the “everyone should learn programming” that became very strong in 2012.
  • Engineered Truth: How To Teach Yourself Code (ft. Quincy Larson) (YouTube, 17 minutes, 26.05.2016): interview with Quincy Larson, a teacher at freeCodeCamp.org. Larson's tips are the following: (1) code every day (use the Pomodoro Technique); (2) hang out with other people who code (they can make you feel accountable to them and thereby keep your motivation going; you can also pair-program with them); (3) the tools really don't matter (it does not matter which programming tool or framework you use to learn to program; skills are transferable); (4) don't worry what to build; focus on how to build it; (5) contribute to open source (legacy code).
  • Dreaded Dev: How I Became a Web Developer in 2 Months and Got a Job! (YouTube, 14 minutes, 09.11.2017): Dreaded Dev learned web development through Treehouse, Udemy, Codeacademy, freeCodeCamp and YouTube, all for less than $50.
  • Dreaded Dev: How You Can Become a Web Developer in 2 Months (YouTube, 14 minutes, 13.11.2017): video about the skills you need to become a web developer. You need to start with HTML and CSS; you should be able to learn these skills with freely available resources. Then see if you can create a website; for example, go to Behance, choose a difficult design and recreate it in HTML and CSS. (You can also do a web search for PSD website mockups.) Then you can move on to JavaScript. There are many free resources about JavaScript, but many of them are outdated or incomplete. So it can help to pay for resources such as the developer track on Treehouse or a Udemy course. This will take longer to master; e.g. a few weeks to master the basics. You should then take a look at Bootstrap, which is often rapid prototyping, or Foundation. Also learn jQuery (possibly before Bootstrap or Foundation). Eventually, you should be able to get through the JavaScript section on freeCodeCamp and solve coding challenges such as the pomodoro clock.
  • Sterkowitz, Andrew: My Story of Being a Self Taught Programmer (Plus Tips and Advice!) (YouTube, 13 minutes, 04.01.2018): story of a former car salesman who started learning software development in early 2014. He got through the book Head First JavaScript Programming in a month, then started creating a portfolio of applications to show to pontential employers. His first JavaScript program was a Tetris implementation. Then he went back to school (autumn – December 2014). After Christmas 2014, he started applying for jobs. He did not get a response from the first 30 or 40 employers he contacted. Then, he replaced his résumé with a bio, and his work experience with the applications he had created; this quickly led to his first programming job. He says that you need to set yourself a definite target date when you start applying for jobs.
