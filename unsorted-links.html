  • [76] How To Learn The Binding Order When Picking Locks (HelpfulLockPicker on YouTube, 12 minutes, 06.09.2017).
    This video explains what binding order is using two acrylic locks and a cutawa lock.   • [188] My Approach to Lock Picking Tension (LockPickingLawyer on YouTube, 16 minutes, 18.06.2016).
    Tension is an underrated aspect of picking. The advice, “use the lightest tension possible” is too simplistic. A tension tool should be stiff, not flexible, so as to transmit feedback from the lock to the picker. Using top-of-the-keyway tension is that it leaves room for the picking tool. Bottom-of-the-keyway tension can be useful in big keyways, so you can use it as a platform for the pick.
    LPL's advice is to use the heaviest tension possible for each particular lock. Heavier tension provides clearer feedback, which makes it more obvious which pins are binding. It also has other benefits. Tension is not static.

  • 5 Things You Should Never Say In a Job Interview (Don Georgevich on YouTube, 13 minutes, 12.02.2019).
    First, never say that you didn't like the boss or the company where you are leaving. Second, don't say that you are willing to do anything; it shows a level of desperation that does not come over well. Instead, talk about what you can do and what value you can bring to the company. Third, when you are asked to talk about yourself, don't say that you are a self starter or a quick learner or that you are result oriented; instead put this in context by talking about how you have done this before and let the interviewer draw the conclusion. Fourth, never say that you don't know how to do something they ask you about; instead, say something like, “I need to think about this for a second” or “I have never done the exact same thing but ...” The reason is that these types of questions are compentency questions and try to find out the fundamentals you know.
