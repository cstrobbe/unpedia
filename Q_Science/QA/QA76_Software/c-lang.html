Editors, IDEs and Other Tools

  • tis-interpreter is a tool that detects subtle bugs in C programs that may not have eye-visible effects when executing the same programs compiled in the traditional way. The tis-interpreter source code is available on GitHub. The repository contains several licence files and source code files such as docgen_ge400.ml mention the GNU Lesser General Public License version 2.1.
  • CScout: CScout is a source code analyzer and refactoring browser for collections of C programs. See also the website CScout: The C Refactoring Browser and the presentation C Code Refactoring: Working with CScout, the C Refactoring Browser by Diomidis Spinellis (FOSDEM 2016), who developed the tool.
  • Splint win32 binaries. Splint is a tool for statically checking C programs for security vulnerabilities and coding mistake. Available under the GPL licence version 2. The code has not been modified since 2007. See also the Splint - Secure Programming Lint at the University of Virginia. (The code is also available on SourceForge and in another GitHub repository.)

C Libraries and Tools

  • Wikipedia: C standard library.
  • Apache Celix: an implementation of the OSGi specification adapted to C and C++. See also: Modularizing C software with Apache Celix by Pepijn Noltes (FOSDEM 2015).
  • GLib: GLib provides the core application building blocks for libraries and applications written in C. It provides the core object system used in GNOME, the main loop implementation, and a large set of utility functions for strings and common data structures.
  • GTK+: a multi-platform toolkit for creating graphical user interfaces. See also the article GTK+ on Wikipedia.
  • liblfds: a portable, license-free, lock-free data structure library written in C.
  • SQLite: open source SQL database engine written in ANSI C.
