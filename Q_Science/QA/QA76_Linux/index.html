  • ExplainShell.com: write down a command-line to see the help text that matches each argument.
  • 8 Bash Shortcuts Every Linux User Should Know (YouTube, 10 minutes, 20.05.2018). This video covers the basics moving around in the shell. The video demonstrates the following keyboard shortcuts: CTRL + C to terminate a shell session or a running shell program or command, CTRL + D to exit the shell session (or to log out of a remote session), tab completion (a single TAB stroke to auto-complete; a double TAB to see list of possible completions), CTRL + L to clear the terminal, using the arrows to navigate command history, CTRL + A to move to the beginning of a command, CTRL + E to move to the end of a command, ALT + F and ALT + B to move forward and backward by one word, respectively, CTRL + R to do a reverse search of the command history. These commands should become part of your muscle memory.

Linux Security

Linux security here covers two broad aspects: making Linux computers and servers more secure (also known as “hardening”) and testing the security of Linux systems. For the latter type of task, there are even dedicated distros that are not intended for regular computer work, for example Kali Linux.

  • LinuxSecurity.com.
  • Tevault, Donald A.: Mastering Linux Security and Hardening. Packt, 2018. (376 pages) ISBN 9781788620307.
    See also the author's video presentation: Mastering Linux Security and Hardening (YouTube, 4 minutes, 10.02.2018).
  • OccupyTheWeb: Linux Basics for Hackers: Getting Started with Networking, Scripting, and Security in Kali. No Starch Press, July 2018. ISBN 978-1-59327-855-7.
  • Rankin, Kyle: Linux Hardening in Hostile Networks: Server Security from TLS to Tor. Pearson Open Source Software Development Series. Addison-Wesley Professional, 2017. (272 pages) ISBN 978-0-13-417326-9.
    This book covers overall security concepts, workstation security, server security, network security, e-mail, DNS, databases, incident response, Tor and SSL/TLS.
  • Turnbull, James: Hardening Linux. Apress, 2005. (584 pages) 978-1-59059-444-5.
    See also the source code for Hardening Linux on GitHub.
  • Cromwell, Bob: The root password is ... irrelevant, Bob Cromwell, 10.03.2018.
  • Cromwell, Bob: How To Harden a Linux or OpenBSD Installation, Bob Cromwell, 10.03.2018.
  • Linux security, Gender and Tech Solutions, last updated in November 2015 (licence: CC-BY-NC-SA). A single-page introduction that covers intrusion detection, rootkit scanners, password management, firewalls, safe browsing, removable storage media, encryption, backups, traffic monitoring, sysctl and auditing.

Specific Distributions

There are Linux distributions for many different types of purposes: everyday desktop computing, embedded computing, deployment on old devices with few resources, etcetera. See for example the following Wikipedia articles: Light-weight Linux distribution, Security-focused operating system, Linux on embedded systems and Linux range of use.

  • The LWN.net Linux Distribution List.
  • Antergos: a rolling release distribution. The purpose of Antergos is to provide a modern, elegant, and powerful operating system based on one of the best Linux distributions available, Arch Linux.
  • antiX Linux: a fast, lightweight and easy to install systemd-free linux live CD distribution based on Debian Stable for Intel-AMD x86 compatible systems. Version 17.2.1 was released in November 2018; version 17.4.1 was released in March 2019.
  • Arch Linux: a lightweight and flexible Linux® distribution that tries to Keep It Simple. Several other distribution are based on Arch, such as Manjaro, Antergos and ArchBang. See also the Arch Linux wiki, which has articles on general recommendations, core utilities, Bash and desktop environment.
  • Bodhi Linux: a lightweight distribution based on Ubuntu LTS releases that uses the Moksha window manager. Version 5.0.0 was released in August 2018.
  • The CentOS Project. CentOS is a distribution based on Red Hat Enterprise Linux (RHEL). The old CentOS-5 Documentation is still available on the website, but for up-to-date documentation, the website refers users to the upstream documentation by Red Hat.
  • ChaletOS is a distribution based on Xubuntu that tries to look similar to Microsoft Windows.
  • Damn Small Linux is a very versatile 50MB mini desktop oriented Linux distribution. See also The Official Damn Small Linux Book (2007).
  • Debian -- The Universal Operating System: one of the purest, oldest and most stable existing Linux distributions, which serves as a basis for many other Linux distributions, such as Ubuntu, tails and Rescatux. Debian is often used for web servers.
    See also The Debian Administrator's Handbook, which also contains a chapter on Becoming a Package Maintainer. People who want to contribute to Debian should consult Debian New Maintainers' Guide.
  • Devuan GNU+Linux: a fork of Debian that uses sysvinit instead of systemd. In spite of its young age (the fork dates from 2014), there are several distributions based on Devuan. See also Jaromil Rojo's presentation Devuan GNU+Linux: where we are, how we got here, where we are going (YouTube, 44 minutes) at FSCONS, Göteborg, November 2016. In June 2018, Bruce Perens wrote on slashdot, I was the second Debian project leader. These days, I prefer to run Devuan, a true Debian derivative engineered the way I would probably have decided to make it. It's efficient and trouble-free. Thanks to the Devuan developers for all of the work! (Devuan is a registered trademark of the Dyne.org foundation, a non-profit free software foundry based in the Netherlands.)
  • elementary OS: A fast and open replacement for Windows and macOS. This distribution is based on Ubuntu LTS.
  • Endless OS: The operating system that comes with everything your family needs.
  • Fedora: a distribution sponsored by Red Hat and maintained by a community. See also the following links:
    • Fedora Magazine.
    • At the end of October 2018, IBM acquired Red Hat.
    • Fedora 29 Tweaking - My Distro Setup Routine (InfinitelyGalactic on YouTube, 19 minutes, 04.12.2018).
      These tweaks assume that you are running Gnome. The tweaks concern the addition of repositories, the installation of Gnome Tweak Tool, turning off Gnome's search facility (in Gnome Tweak Tool), the installation of Microsoft fonts (including the new font rendering engine that Microsoft open-sourced), screen calibration, GUI tweaks (including themes and adding Gnome's Dash to Panel), installing Timeshift as a backup tool, and more. At the end of the video, Gnome looks significantly differently from the default installation.
  • FluXuan Linux: Devuan-based distribution that uses Fluxbox as window manager and LightDM as display manager.
  • Gentoo Linux: a Linux distribution that focuses on configurability and performance and that is not based on another distribution. See also the InstallGentoo Wiki.
    The project also maintains Hardened Gentoo, which offers multiple additional security services on top of the well-known Gentoo Linux installation. Hardened Gentoo serves as the basis for CLIP OS, which is maintained by the National Cybersecurity Agency of France (ANSSI).
  • gNewSense: a fully free software GNU/Linux distribution sponsored by the Free Software Foundation (FSF). Version 4.0, based on Debian 7, was released in May 2016; support ended in May 2018.
  • heads: a live distribution based on Devuan with a focus on privacy and security. (The operating system is an alternative to tails, which is based on Debian.)
  • Kali Linux | Penetration Testing and Ethical Hacking Linux Distribution: this is a specialised distribution that is not intended for everyday use but specifically for penetration testing and security-related tasks. See also Why is Kali Linux so hard to set up? Why won't people help me? (a question asked on Unix & Linux Stack Exchange).
  • KDE neon is a Linux distribution created by the KDE community, based on a long-term support release of Ubuntu.
  • Linux From Scratch (LFS) is a project that provides you with step-by-step instructions for building your own custom Linux system, entirely from source code. It is not actually a Linux distribution but a set of books that guide you through the process of building a Linux system.
  • Linux Lite: a lightweight Linux distribution that describes itself as a gateway operating system that was created to make the transition from Windows to a linux based operating system, as smooth as possible. Version 4.2 was released in October 2018.
  • Linux Mint: a distribution that comes in versions based on either Ubuntu (Ubuntu Mint) or Debian (LMDE). The distribution is available with a number of desktop environments, including Cinnamon (a fork of Gnome Shell), MATE, KDE and Xfce. In Linux Mint, these desktops have a task bar and a start menu, just like Microsoft Windows. See also Why Is Linux Mint So Popular? - the History of Linux Mint (InfinitelyGalactic on YouTube, 16 minutes, 20.11.2018).
  • Lubuntu: a Linux distribution based on Ubuntu that uses the Lightweight Qt Desktop Environment (LXQt). See also the Lubuntu Manual. Version 18.10 was released in October 2018.
  • Mageia: a Linux distribution that started as a fork of Mandriva Linux. The legal entity behind it—Mageia.Org—is based in Paris, France.
  • Manjaro Linux: a rolling distro based on Arch, but much more user friendly. See also the Manjaro Wiki. Since Manjaro allows installing packages from the Arch User Repository (AUR), see also the article “Arch User Repository” in the Manjaro Wiki.
    On Distrowatch in 2018, Manjaro is extremely popular (first place for several months and with a long distance before Mint). See Why Is Manjaro So Popular? (InfinitelyGalactic on YouTube, 19 minutes, 18.09.2018).
  • MX Linux is a midweight OS designed to combine an elegant and efficient desktop with simple configuration, high stability, solid performance and medium-sized footprint. See InfinitelyGalactic's YouTube video: MX Linux 17.1 Review - The dream distro? (14 minutes, 06.11.2018).
  • Nitrux: Nitrux is a Linux distribution based on Ubuntu suitable for laptops and desktop computers. (…) Nitrux is suitable for newcomers to Linux as well as *nix experienced users. Nitrux uses KDE Plasma 5 and KDE Applications; we also use our in-house software suite Nomad Desktop adding to the user experience. Nitrux uses the NX Desktop (formerly Nomad Dekstop), a customization layer for Plasma 5.
    According to InfinitelyGalactic's YouTube video Welcome to the Future?! - Nitrux OS Review (9 minutes, 23.10.2018) Nitrux looks very interesting, partly due to its attention to design, but may not quite be ready for prime time yet.
  • NuTyX GNU/Linux. This distribution was inspired by Linux From Scratch
  • OpenMandriva: a Linux distribution that is a continuation of Mandriva Linux, which resulted from the fusion of Mandrake Linux and Conective Linux.
  • openSUSE a Linux distribution that is currently available in two versions: Tumbleweed (a rolling-release version) and Leap. It has some well-known tools: YaST, Zypper and openSUSE Build Service (a public instance of Open Build Service).
  • Parrot OS: a security-oriented distribution based on Debian. See also the video Get Started with Parrot Security OS on Your Computer [Tutorial] by Null Byte on YouTube (14 minutes, 11.05.2019).
  • PCLinuxOS: a free easy to use Linux-based Operating System for x86_64 desktops or laptops. It is available with either the KDE desktop or the MATE desktop. The latest version was released in June 2018.
  • Porteus: a lightweight Linux distribution based on Slackware and optimized to run from CD, USB flash drive, hard drive, or other bootable storage media. It is available in both a 32-bit and a 64-bit version. Version 4.0 was released in April 2018.
  • Peppermint: a distribution based on Ubuntu that uses desktop components from several desktop environments (Cinnamon and XFCE). It is available in 32-bit and 64-bit versions. (Peppermint used to have an LXDE desktop; Peppermint 9 continued the migration away from LXDE.)
  • Red Hat Enterprise Linux: a commercial distribution offered by Red Hat. See also Product Documentation - Red Hat Customer Portal.
  • RoboLinux is a Linux distribution that makes it easy for Windows users to run programs written for Microsoft Windows. It includes StealthVM, in which you should be able to run Windows programs without the risk of virus infections.
  • Sabayon Linux is a beginner-friendly Gentoo-based open-source Linux distribution. The developers say that the offer a bleeding edge operating system that is both stable and reliable. There are several editions of Sabayon, depending on the desktop you prefer: GNOME, KDE, Xfce, Mate and Fluxbox.
  • Scientific Linux: a distribution based on Red Hat Enterprise Linux.
  • The Slackware Linux Project: the oldest existing Linux distribution. It is not updated as frequently as Ubuntu or Debian and has a reputation of being aimed at advanced Linux users. For example, there is no “automatic” dependency resolution and LILO is still being used as a boot loader. See also SlackBuilds.org, sbopkg : the slackbuilds.org package browser and The Revised Slackware Book Project (last updated in 2012).
  • Solus: an operating system that is designed for home computing. This is an independent Linux distributions that is not based on existing one.
  • Tails: The Amnesic Incognito Live System: a Linux distribution based on Debian and designed for online anonymity and privacy. It is typically run as a “live” operating system.
  • Ubuntu: one of the best-known Linux distributions. Ubuntu is based on Debian and has two releases per year. A long-term support version (LTS) is released every two years; these versions are supported for five years.
    Vaughan-Nichols, Steven J.: Mark Shuttleworth reveals Ubuntu 18.04 will get a 10-year support lifespan, ZDNet, 15.11.2018.
  • Ubuntu Studio: Ubuntu Studio is a free and open source operative system, and an official flavor of Ubuntu. Ubuntu Studio is the most widely used multimedia orientated GNU/Linux distribution in the world. It comes preinstalled with a selection of the most common free multimedia applications available, and is configured for best performance for the Ubuntu Studio defined workflows: Audio, Graphics, Video, Photography and Publishing.
  • Zorin OS: a Linux distribution that offers desktops that are similar to Windows and MacOS with the goal of lowering the threshold for Windows and MacOS users who want to transition to Linux.
    Zorin OS is available in several editions: Zorin OS Education (for students; free), Zorin OS Lite (for older computers; free), Zorin OS Core (free, but a donation would be appreciated), Zorin OS Business (€ 15 per computer) and Zorin OS Ultimate (€ 19). Only the Ultimate edition has the ability to display a desktop like that of MacOS.
    For reviews, see for example Zorin OS 12.1 Review : Best Windows Replacement? (YouTube, 7 minutes, 31.03.2017).

Other operating systems that are not Linux distributions (temporarily listed here):

  • Chromium OS: an open-source project that aims to build an operating system that provides a fast, simple, and more secure computing experience for people who spend most of their time on the web. See Google's announcement from November 2009: Releasing the Chromium OS open source project.
  • GNU/Hurd: the GNU project's replacement for the Unix kernel. According to the status page it is not yet ready for production use. For those who wish to test it, the best option is Debian GNU/Hurd.
    See also the video Installing the GNU hurd in a vm 2017 by The GNU Guy (YouTube, 29 minutes, 25.10.2017) and Michael Banck's presentation at DebConf10 Debian GNU/Hurd: Past, Presence and Future (YouTube, 44 minutes; 2 August 2010).
  • Qubes OS: A reasonably secure operating system: strictly speaking not a Linux distribution but a Unix-like operating system that relies, among other things, on virtualisation and full disk encryption to create a more secure operating system. One should consult the Hardware Compatibility List (HCL) before installing it. The Qubes OS project was founded by Joanna Rutkowska, who is also the founder of Invisible Things.
    For a high-level overview, see for example Matty McFatty's presentation I used the reasonably-secure Qubes OS for 6 months and survived (YouTube, 19 minutes, 09.01.2018). For a very short introduction, see the video The Qubes OS, Approved by Edward Snowden (YouTube, 6 minutes, 05.11.2017). See also Qubes OS - The Security-Focused OS Used by Snowden (YouTube, 28 minutes, 17.12.2016): an interview with Andrew Wong, the Qubes OS Project's community manager.
    For an older introduction, see Joanna Rutkowska's presentation Towards (reasonably) trustworthy x86 laptops (YouTube, 63 minutes, 27.12.2015).
  • Redox OS: a Unix-like Operating System written in Rust, aiming to bring the innovations of Rust to a modern microkernel and full set of applications. Rust is a systems programming language that is designed to be safer than C and C++. See also the Redox Gitlab repositories, the Redox GiHub repositories and David Teller's FOSDEM 2017 presentation Redox OS: A safety-first microkernel developed in Rust (YouTube, 42 minutes, 07.03.2018; Creative Commons licence).
  • Haiku: Haiku is an open-source operating system that specifically targets personal computing. Inspired by the BeOS, Haiku is fast, simple to use, easy to learn and yet very powerful. Haiku R1/beta1 was released on 28 September 2018. See also John Paul's brief review After 16 Years of Development, The First Beta of Haiku is Finally Here (It's FOSS, 11.10.2018) and Christopher Barnatt's first impressions: Haiku Alternative Operating System (ExplainingComputers on YouTube, 12 minutes, 28.10.2018).
  • FreeBSD. See also the FreeBSD Handbook, the book Absolute FreeBSD: The Complete Guide to FreeBSD, third edition, by Michael W. Lucas (No Starch Press, 2018; ISBN 9781593278922), the book Design and Implementation of the FreeBSD Operating System, second edition, by Marshall Kirk McKusick, George V. Neville-Neil and Robert N.M. Watson (Addison-Wesley Professional, 2014, ISBN 978-0-321-96897-5), Info Q's review and interview with Marshall Kirk McKusick (13.08.2015) and the FreeBSD Foundation.
  • DragonFly BSD: forked from FreeBSD in June 2003. It has its own filesystem known as HAMMER.
  • TrueOS: TrueOS is a powerful, free, and open source server operating system that will help you get the job done at home or at work. Formerly known as PC-BSD, this operating systems is based on FreeBSD, claims to be very secure and uses the OpenZFS file system. Since 2014, TrueOS has its own desktop environment: Lumina.
    See also the video TrueOS: Linux or Windows Alternative (ExplainingComputers on YouTube, 13 minutes, 13.05.2018) and the blog post TrueOS (FreeBSD) tested on real hardware ThinkPad T410 (Panorama as a circle, 06.08.2018), which warns that Intel video drivers don't work in this operating system.
  • HardenedBSD: Founded in 2014 by Oliver Pinter and Shawn Webb, HardenedBSD is a security-enhanced fork of FreeBSD. The HardenedBSD Project is implementing many exploit mitigation and security technologies on top of FreeBSD. See also The HardenedBSD Project on GitHub.
  • OpenBSD. See also the books Absolute OpenBSD: Unix for the Practical Paranoid, second edition, by Michael W. Lucas (No Starch Press, 2013; ISBN 978-1-59327-476-4) and The Book of PF: A No-Nonsense Guide to the OpenBSD Firewall, third edition, by Peter N. M. Hansteen (No Starch Press, 2014; ISBN 978-1-59327-589-1).
  • NetBSD. Version 8.0 was released in July 2018.
  • GhostBSD: built on top of FreeBSD and geared at dekstop users. It is available in a MATE edition and an XFCE edition.

Other, smaller efforts:

  • openindiana – Community-driven illumos Distribution. See for example the video My First Time Looking At OpenIndiana (DistroTube on YouTube, 25 minutes, 10.09.2018), which looks at the “Hipster”, wich uses the Mate desktop.
  • illumos: continuation of openSolaris, which was discontinued by Oracle after the acquisition of Sun Microsystems. It is compatible with Solaris at the binary level.
  • FreeDOS: an operating system that attempts to recreate Microsoft DOS. The project started in 1994. FreeDOS 1.3 RC1 was released in February 2019.
  • ReactOS: a free and open-source operating system that is designed to provide binary compatibility with Windows. Version 0.4.11 alpha was released in March 2019.
  • Menuet OS: a pre-emptive, real-time and multiprocessor Operating System in development for the PC written entirely in 32/64 bit assembly language. It is very light and rapid.
  • KolibriOS: an operating system that forked from Menuet OS in 2004. It is also very small and fast.
  • V7/x86 - x86 port of UNIX V7 (released in 2012; maintained by Robert Nordier from South Africa).
  • Syllable Desktop: an operating system forked from the discontinued AtheOS in 2002. AtheOS had attempted to recreate the experience of Amiga OS.
  • Visopsys | Visual Operating System: an alternative operating system for PC compatible computers. Version 0.83 was released on 09.08.2018.
  • JNode: an operating system written in Java. Version 0.2.8 was released in 2009.
