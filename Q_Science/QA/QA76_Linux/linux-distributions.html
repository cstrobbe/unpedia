General Information About Linux Distributions

The large majority of existing Linux distributions are based on a small set of other, usually independent, distributions, namely Debian, openSUSE, Fedora, Arch, Gentoo or Slackware. Many are based on Ubuntu, which is itself based on Debian. Of course, there are other independent distributions, such as Solus, Dragora and Void Linux.

Most of the distributions listed on this page are for general purpose use, but some of these serve a specific purpose. In addition to everyday desktop computing, there are also Linux distributions for embedded computing, deployment on old devices with few resources, security testing, etcetera.
For example, the following distributions focus on specific audiences, tasks or hardware characteristics:

  • making it easy for Windows users to migrate to Linux: Chalet OS, Linux Mint, Ubuntu Kylin and Zorin OS;
  • system rescue tasks: Finnix, GParted Live, Grml, Rescatux and SystemRescueCd;
  • preserving online privacy and anonymity: Tails, Kodachi, heads, Septor and Whonix;
  • digital forensics, penetration testing and other security-related tasks: Kali Linux, Parrot OS, BlackArch and BackBox;
  • older hardware: Bodhi Linux, Porteus, Puppy Linux, Slax Linux, SliTaz GNU/Linux, Tiny Core Linux, ToriOS and ttylinux.

The website Distrochooser presents sixteen questions about how you prefer to use operating systems and applications, and, based on your answers, presents a list of Linux distribution that fit your criteria. For a similar webpage in French, see Choisis ton GNU.
See also the following Wikipedia articles: Light-weight Linux distribution, Security-focused operating system, Linux on embedded systems and Linux range of use.

Individual Distributions

Linux distributions are listed in alphabetical order.

The list below also tracks whether a distribution provides documentation on accessibility and assistive technologies, or has an accessibility team or contributors focusing on these aspects of HCI. In the case of commercial distributions, this may include mentioning whether there is a Voluntary Product Accessibility Template or VPAT for the distribution. A VPAT is a table used to document conformance withSection 508 of the U.S. Rehabilitation Act of 1973. VPATs are usually created by software companies that wish to sell software to the U.S. federal government.
For general information about accessibility on Linux, see Linux Accessibility Resources.

4MLinux

4MLinux is a is a small, independent, general-purpose Linux distribution with a strong focus on the following four "M" of computing: Maintenance (system rescue Live CD), Multimedia (full support for a huge number of image, audio and video formats), Miniserver (DNS, FTP, HTTP, MySQL, NFS, Proxy, SMTP, SSH, and Telnet), Mystery (meaning a collection of classic Linux games). See also the 4MLinux forum on LinuxQuestions.org.
4MLinux 30.1 was released on 20.10.2019.

The 4MLinux website does not provide any information on accessibility or compatibility with assistive technologies.

Alpine

Alpine Linux describes itself as a security-oriented, lightweight Linux distribution based on musl libc and busybox. It uses musl libc and BusyBox as a replacement for some traditional GNU utilities, a package manager called apk, the OpenRC init system (instead of systemd) and was designed with security in mind. It can be used in embedded devices. For documentation and support, see the Alpine Linux Wiki and the Alpine Linux Community.

The Alpine Linux website does not provide any information on accessibility or compatibility with assistive technologies, except for mentioning the KDE application kde-applications-accessibility on its KDE wiki page.

Arch Linux

Arch Linux: a lightweight and flexible Linux® distribution that tries to Keep It Simple. Several other distribution are based on Arch, such as Manjaro, Antergos and ArchBang. For documentation and support, see the Arch Linux wiki, the Arch Linux forums and the Arch Linux mailing lists.
The wiki has articles on general recommendations, core utilities, Bash and desktop environment.
Also worth mentioning: pacapt is a pacman-like wrapper for many package managers.

The Arch installation process takes a bit of time and requires more than average knowledge of Linux. ArcoLinux

ArcoLinux is an Arch-based Linux distribution developed by a team led by Erik Dubois from Belgium. The distribution comes in three major editions:

  • the regular ArcoLinux edition, which provides a full featured iso containing a ton of software and 3 desktops Xfce, Openbox and i3,
  • ArcoLinuxD, which is a minimal iso dropping you in the command line, so you can learn about Linux by building your own system on this basis, and
  • ArcoLinuxB, which will give the user the power to actually build his own iso.

The motto of ArcoLinux is Learn, Have Fun and Enjoy. One of the distribution's goal is for users to learn: Come for the learning experience and improve your Linux skills through the many articles and Youtube articles. Invest your time in ArcoLinux and we will invest in you. We have a 1-2 year learning plan if you like to improve your linux skills.
For documentation and support, see the help section on ArcoLinux.com, ArcoLinux.info, ArcoLinuxB.com, ArcoLinuxD.com, the ArcoLinux -D -B Forum, the ArcoLinux blog and Erik Dubois' YouTube channel.

The name ArcoLinux resulted from a rebranding of ArchMerge in 2018.

Clear Linux is an open source, rolling release Linux distribution optimized for performance and security, from the Cloud to the Edge, designed for customization, and manageability. Users will be interested in the Clear Linux Forums, the Clear Linux documentation, the Clear Linux* Project Docs and the Clear Linux Store. Clear Linux should not be confused with Clear OS.

Cucumber Linux

Cucumber Linux aims to provide a Linux distribution that is usable as an every day, general purpose operating system. It aims to do this in as minimalistic a way as possible and in a way that follows the Unix Philosophy. It is an independent Linux distribution that puts a strong emphasis on security. The developers also point out that since Systemd blatantly violates all 17 of Eric Raymond's Unix Rules, Cucumber Linux will always strive remain a Systemd free distribution.

Debian

Debian -- The Universal Operating System: one of the purest, oldest and most stable existing Linux distributions, which serves as a basis for many other Linux distributions, such as Ubuntu, tails and Rescatux. Debian is often used for web servers.

For documentation and support, see the Debian wiki, which provides information on a wide range of subjects that are relevant to this distribution, DebianUserForums.org and debianforum.de (in German).
Users coming from Ubuntu or Linux Mint should also read Don't Break Debian in the Debian wiki.
See also The Debian Administrator's Handbook, which also contains a chapter on Becoming a Package Maintainer. People who want to contribute to Debian should consult Debian New Maintainers' Guide.
The latest version, Debian 10 "buster", was released in September 2019.

Debian is one of a relatively small number of Linux distribution projects that have an accessibility project: Debian-Accessibility: Debian-Accessibility is an internal project to develop Debian into an operating system that is particularly well fit for the requirements of people with disabilities. The goal of Debian-Accessibility is a completely accessible system which offers users with disabilities the highest possible amount of independence, built completely on free software. The wiki page accessibility-maint contains essential accessibility information for package maintainers.
Outside the official Debian project, Simon Kainz created check-a11y, a tool that automatically checks the status of accessibility support in various desktop environments that are supported by Debian.

The wiki page openSUSE:Statistics contains the following information related to privacy (accessed 10.11.2019):

Counting IP addresses has the disadvantage that many machines are hidden behind proxy servers or masquerading routers. In order to get a more accurate view on the number of installations the openSUSE project instead counts unique Zypper UUID signatures hitting download.opensuse.org. This UUID string is a random label that is created during an installation. The openSUSE project does not use this id to track anything but to count. There is no way to relate this random string to a user or machine.

The UUID is stored in the file /var/lib/zypp/AnonymousUniqueId. In order to not have an installation counted for the statistics the file can be removed or emptied. Alternatively online repositories can be switched to use a specific mirror server instead of the automatic redirector on download.opensuse.org.

openSUSE is one of the few Linux distributions that provide documentation on assistive technologies. See Assistive Technologies (GNOME User Guide)). There are also blog posts that discuss or mention accessibility: Improving low-vision accessibility of the installer (07.10.2016) and Highlights of YaST development sprint 26 (20.10.2016, mentioning work to make the installer accessible to low-vision users). Zorin OS

Zorin OS is an Ubuntu-based distribution that offers desktops that are similar to Windows and MacOS with the goal of lowering the threshold for Windows and MacOS users who want to transition to Linux. The distribution is developed by Zorin OS Technologies in Dublin, Ireland.
Zorin OS is available in several editions: Zorin OS Education (for students; free), Zorin OS Lite (for older computers; free), Zorin OS Core (free, but a donation would be appreciated), Zorin OS Business (€ 15 per computer) and Zorin OS Ultimate (€ 19). Only the Ultimate edition has the ability to display a desktop like that of MacOS.
For documentation and support, see the Zorin OS Help page and the Zorin Group Forum.

In the YouTube video Building a Linux OS for the Masses - Artyom Zorin Interview (54 minutes, 12.09.2019), InfinitelyGalactic talks with Artyom Zorin, co-founder of Zorin OS. Topics include the early goals of Zorin OS (making Linux user friendly for people who wanted to switch over from Windows or Mac OS to Linux), taking on feedback from diehard Linux users (they looked at it very closely and noticed that different types of users had different types of feedback; Zorin OS was not only meant for new users), the tension between creating a familiar user experience versus an intuitive user experience (Zorin uses a blend of both: some user experience elements are similar across operating systems and for good reasons, and if a new user encounters a usability problem, that may put them off Linux. However, the developers don't want Zorin to look exactly like Windows), releases that made a bigger impact (version 3, version 8 when Windows XP was losing support, and version 15), the expected user base (for example, the Italian city of Vicenza switched over from Windows to Zorin OS in 2016), monetising Linux projects and keeping them sustainable if they are not funded by a big company such as Intel, partnerships with hardware vendor in order to get Zorin OS preinstalled on computers, and the future of Zorin OS as companies will be looking for a replacement for Windows 7.

Zorin OS does not provide information related to accessibility or assistive technology, nor does the company provide a VPAT for the operating system. In May 2014, there was a discussion about Accessibility Tools in Future Releases [of Zorin OS]?, but there was no response from the developer team. (In 2013, there was some discussion about accessibility on the Other OS's Discussion Board.)

For reviews, see for example Zorin OS 12.1 Review : Best Windows Replacement? (YouTube, 7 minutes, 31.03.2017).

Discontinued Linux Distributions

Over the years, many distributions have come and gone. They include Softlanding Linux System, Mandrake (and its successor Mandriva), Yoper Linux, Pear Linux, Damn Small Linux and many others. This section of the page lists only distribution that were discontinued after the creation of the above list. They may be removed later, especially if their websites disappear.

Antergos

Antergos was a rolling release distribution based on Arch. The purpose of Antergos is to provide a modern, elegant, and powerful operating system based on one of the best Linux distributions available, Arch Linux. In May 2019, the Antergos project announced its development would end.

  • Dustin: Antergos Linux Project Ends, Antergos website (archived), 21.05.2019.
  • Verma, Adarsh: Antergos Linux Project Is Dead — Here Are Some Alternatives To Try, FOSSBytes, 22.05.2019. As they rightly mentioned, it would have been a “huge disservice to the community” if they had chosen to continue the Antergos project in a similar manner. The article recommends the following alternatives: Manjaro, ArcoLinux, ArchBang Linux, ArchLabs Linux, Artix Linux and SwagArch Linux.

Non-Linux Operating Systems

Other operating systems that are not Linux distributions (temporarily listed here):

  • Chromium OS: an open-source project that aims to build an operating system that provides a fast, simple, and more secure computing experience for people who spend most of their time on the web. See Google's announcement from November 2009: Releasing the Chromium OS open source project.
  • GNU/Hurd: the GNU project's replacement for the Unix kernel. According to the status page it is not yet ready for production use. For those who wish to test it, the best options are Debian GNU/Hurd and Arch Hurd.
    See also the video Installing the GNU hurd in a vm 2017 by The GNU Guy (YouTube, 29 minutes, 25.10.2017) and Michael Banck's presentation at DebConf10 Debian GNU/Hurd: Past, Presence and Future (YouTube, 44 minutes; 2 August 2010).
  • Qubes OS: A reasonably secure operating system: strictly speaking not a Linux distribution but a Unix-like operating system that relies, among other things, on virtualisation and full disk encryption to create a more secure operating system. One should consult the Hardware Compatibility List (HCL) before installing it. The Qubes OS project was founded by Joanna Rutkowska, who is also the founder of Invisible Things.
    For a high-level overview, see for example Matty McFatty's presentation I used the reasonably-secure Qubes OS for 6 months and survived (YouTube, 19 minutes, 09.01.2018). For a very short introduction, see the video The Qubes OS, Approved by Edward Snowden (YouTube, 6 minutes, 05.11.2017). See also Qubes OS - The Security-Focused OS Used by Snowden (YouTube, 28 minutes, 17.12.2016): an interview with Andrew Wong, the Qubes OS Project's community manager.
    For an older introduction, see Joanna Rutkowska's presentation Towards (reasonably) trustworthy x86 laptops (YouTube, 63 minutes, 27.12.2015).
  • Redox OS: a Unix-like Operating System written in Rust, aiming to bring the innovations of Rust to a modern microkernel and full set of applications. Rust is a systems programming language that is designed to be safer than C and C++. See also the Redox Gitlab repositories, the Redox GiHub repositories and David Teller's FOSDEM 2017 presentation Redox OS: A safety-first microkernel developed in Rust (YouTube, 42 minutes, 07.03.2018; Creative Commons licence).
  • Haiku: Haiku is an open-source operating system that specifically targets personal computing. Inspired by the BeOS, Haiku is fast, simple to use, easy to learn and yet very powerful. Haiku R1/beta1 was released on 28 September 2018. See also John Paul's brief review After 16 Years of Development, The First Beta of Haiku is Finally Here (It's FOSS, 11.10.2018) and Christopher Barnatt's first impressions: Haiku Alternative Operating System (ExplainingComputers on YouTube, 12 minutes, 28.10.2018).
  • FreeBSD. For documentation and support, see the FreeBSD Handbook, the book Absolute FreeBSD: The Complete Guide to FreeBSD, third edition, by Michael W. Lucas (No Starch Press, 2018; ISBN 9781593278922), the book Design and Implementation of the FreeBSD Operating System, second edition, by Marshall Kirk McKusick, George V. Neville-Neil and Robert N.M. Watson (Addison-Wesley Professional, 2014, ISBN 978-0-321-96897-5), Info Q's review and interview with Marshall Kirk McKusick (13.08.2015) and the FreeBSD Foundation.
  • DragonFly BSD: forked from FreeBSD in June 2003. It has its own filesystem known as HAMMER.
  • TrueOS: TrueOS is a powerful, free, and open source server operating system that will help you get the job done at home or at work. Formerly known as PC-BSD, this operating systems is based on FreeBSD, claims to be very secure and uses the OpenZFS file system. Since 2014, TrueOS has its own desktop environment: Lumina.
    See also the video TrueOS: Linux or Windows Alternative (ExplainingComputers on YouTube, 13 minutes, 13.05.2018) and the blog post TrueOS (FreeBSD) tested on real hardware ThinkPad T410 (Panorama as a circle, 06.08.2018), which warns that Intel video drivers don't work in this operating system.
  • HardenedBSD: Founded in 2014 by Oliver Pinter and Shawn Webb, HardenedBSD is a security-enhanced fork of FreeBSD. The HardenedBSD Project is implementing many exploit mitigation and security technologies on top of FreeBSD. For documentation, see the The HardenedBSD Project on GitHub.
  • OpenBSD. See also the books Absolute OpenBSD: Unix for the Practical Paranoid, second edition, by Michael W. Lucas (No Starch Press, 2013; ISBN 978-1-59327-476-4) and The Book of PF: A No-Nonsense Guide to the OpenBSD Firewall, third edition, by Peter N. M. Hansteen (No Starch Press, 2014; ISBN 978-1-59327-589-1).
  • NetBSD. Version 8.0 was released in July 2018.
  • GhostBSD: built on top of FreeBSD and geared at dekstop users. It is available in a MATE edition and an XFCE edition.

Other, smaller efforts:

  • openindiana – Community-driven illumos Distribution. See for example the video My First Time Looking At OpenIndiana (DistroTube on YouTube, 25 minutes, 10.09.2018), which looks at the “Hipster”, wich uses the Mate desktop.
  • illumos: continuation of openSolaris, which was discontinued by Oracle after the acquisition of Sun Microsystems. It is compatible with Solaris at the binary level.
  • SmartOS combines the capabilities you get from a lightweight container OS, optimized to deliver containers, with the robust security, networking and storage capabilities you’ve come to expect and depend on from a hardware hypervisor. The operating system is developed by Joyent, a company that focuses on cloud computing.
  • FreeDOS: an operating system that attempts to recreate Microsoft DOS. The project started in 1994. FreeDOS 1.3 RC1 was released in February 2019.
  • ReactOS: a free and open-source operating system that is designed to provide binary compatibility with Windows. Version 0.4.11 alpha was released in March 2019.
  • Menuet OS: a pre-emptive, real-time and multiprocessor Operating System in development for the PC written entirely in 32/64 bit assembly language. It is very light and rapid.
  • KolibriOS: an operating system that forked from Menuet OS in 2004. It is also very small and fast.
  • V7/x86 - x86 port of UNIX V7 (released in 2012; maintained by Robert Nordier from South Africa).
  • Syllable Desktop (once hosted as www.syllable.org) was an operating system forked from the discontinued AtheOS in 2002. AtheOS had attempted to recreate the experience of Amiga OS.
    According to DistroWatch, Syllable has been discontinued. There has been almost no activity on the syllable-developer mailing list since 2010.
  • Visopsys | Visual Operating System: an alternative operating system for PC compatible computers. Version 0.83 was released on 09.08.2018.
  • JNode: an operating system written in Java. Version 0.2.8 was released in 2009.
