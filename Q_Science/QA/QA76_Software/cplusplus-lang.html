Editors and IDEs

  • Code::Blocks - The open source, cross platform, free C, C++ and Fortran IDE. The Code::Blocks wiki contains information on topics such as compiling errors and debugging (The Code::Blocks source code is available on SourceForge.)
  • Falcon C++ is an open source C++ IDE.
  • Goetz, Lawrence; Langsam, Yedidyah; Raphan, Theodore: Code Blocks (no date).
  • Gomila, Laurent: Test de l'EDI C/C++ Code::Blocks, Developpez.com, 05.09.2005.
  • Dev-C++ (sometimes known as Orwell Dev-C++) is a free, portable, fast and simple C/C++ IDE. The Dev-C++ Blog was last updated in 2015.
  • wxDev-C++ is an open source extension of Dev-C++ that helps you to create dialogs and frames for wxWidgets visually using a form designer.
    wxDev-C++ Portable claims to be the portable version of wxDev-C++. The repository has not been updated since 2015.

C++ Libraries and Tools

  • Awesome C++: A curated list of awesome C/C++ frameworks, libraries, resources, and shiny things, maintained as a community effort.
  • Boost C++ Libraries: free peer-reviewed portable C++ source libraries. (See also Boost (C++ libraries) on Wikipedia.)
    Schäling, Boris: The Boost C++ Libraries: an online book available under Creative Commons Attribution - NonCommercial - NoDerivatives 4.0 International License.
  • Shogun: an open-source machine learning library that offers a wide range of efficient and unified machine learning methods. Licence: GPL 3.0. (See also Shogun (toolbox) on Wikipedia.)
  • mlpack: an intuitive, fast, scalable C++ machine learning library. Licence: BSD-3-Clause licence.
  • POCO C++ Libraries: Modern, powerful open source C++ class libraries for building network- and internet-based applications that run on desktop, server, mobile and embedded systems. Licence: Boost Software License. (See also POCO C++ Libraries on Wikipedia.)
  • VISlib: C++ class library to support development of scientific visualisation applications for Microsoft Windows and Linux operating systems. Licence: BSD licence.
  • LEMON: a C++ template library providing efficient implementations of common data structures and algorithms with focus on combinatorial optimization tasks connected mainly with graphs and networks. Licence: Boost Software License 1.0. (See also LEMON (C++ library) on Wikipedia.)
  • Crypto++ Library: a free C++ class library of cryptographic schemes. Licence: Boost Software License 1.0. (See also Crypto++ on Wikipedia.)
  • Embarcadero: Free C++ Compiler (Embarcadero BCC32C C++ Compiler, with support for C++ 11).
  • Nana C++ Library is an open source modern C++ GUI library. The library currently supports Microsoft Windows and GNU Linux (X11) and has experimental support for Mac OSX. See also the Nana C++ Library Documentation.
