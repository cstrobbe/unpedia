Computer Software

Learning How to Develop Software

  • Felleisen, Matthias; Findler, Robert Bruce; Flatt, Matthew; Krishnamurthi, Shriram: How to Design Programs: An Introduction to Programming and Computing. Second edition. MIT Press, February 2018. (792 pages) ISBN 9780262534802.
    See also the HTML version of the first edition (2001, 2003) and the HTML version of the second edition (2018). This textbook is so well-known that there is even a Wikipedia article about it. (The print version of the first edition, published in 2001, is still listed by MIT Press.)
  • SICP: Structure and Interpreation of Computer Programs (2004): YouTube playlist containing 24 videos (presentations) from a course taught by Professor Eric Grimson at MIT. (Copyright © 2004 Massachusetts Institute of Technology. These videos are licensed under a Creative Commons Attribution-NonCommercial-ShareAlike 2.0 Generic License.)
  • Fish, Shlomi: Thoughts about the Best Introductory Language, Shlomi Fish’s personal website, April 2008.
  • Boothe, Andy: Learning to Program, Data, Code and Conversation, 12.04.2012.
  • Bricklin, Dan: Why Johnny can't program, Dan Bricklin's Web Site, 15.08.2002.
  • Spraul, V. Anton: Think Like a Programmer: An Introduction to Creative Problem Solving. No Starch Press, 2012. (256 pages) ISBN 978-1-59327-424-5.
    Code examples in this book use C++, but the concepts also apply to other programming languages. See also V. Anton Spraul's YouTube channel.
  • Downey, Allen; Monje, Nicholas: How to Think Like a (Functional) Programmer. Green Tea Press; available online. This book is based on Think Python but uses the OCaml language.
  • Engineered Truth: How To Teach Yourself Code (ft. Quincy Larson) (YouTube, 17 minutes, 26.05.2016): interview with Quincy Larson, a teacher at freeCodeCamp.org. Larson's tips are the following: (1) code every day (use the Pomodoro Technique); (2) hang out with other people who code (they can make you feel accountable to them and thereby keep your motivation going; you can also pair-program with them); (3) the tools really don't matter (it does not matter which programming tool or framework you use to learn to program; skills are transferable); (4) don't worry what to build; focus on how to build it; (5) contribute to open source (legacy code).
  • Shah, Dhawal: 620+ Free Online Programming & Computer Science Courses You Can Start in March, freeCodeCamp on Medium, 11.03.2018.
  • Spence, Scott: How to make time for 100 Days Of Code, freeCodeCamp on Medium, 11.03.2018 (7-minute read).

Learning Web Development

  • freeCodeCamp: a site where you can learn to program for free. The topics covered on the site are HTML5, CSS3, JavaScript, databases, Git & GitHub, Node.js, React.js and D3.js. The site focuses on languages and technologies that are used for web development.
  • Codeacademy: a site that offers both free and paid-for software development courses.
  • The Odin Project: a site that offers courses that are relevant to web development: Web Development 101, Ruby, Ruby on Rails, HTML5 & CSS 3 and JavaScript.

Learning C#

  • code easy.net: An interactive, online platform for beginners to learn the C# programming language.
  • Sanfoundry (Manish Bhojasia): C# Programming Examples: a site with many C# examples and their output.

Learning Various Other Programming Languages

  • Code School: a site with programming courses for HTML & CSS, JavaScript, Ruby, Elixir, Android, PHP, .NET, Python, iOS, Git and databases.
  • Exercism: a site with programming problems for many languages, including Clojure, C#, C, C++, EcmaScript (JavaScript), Go, Java, Haskell, Lua, Common Lisp and Kotlin.
  • Codewars: a site with programming challenges for languages such as Clojure, C++, C#, Dart, Elixir, Go, Java, JavaScript, Haskell, PHP, Python and Ruby.
  • SoloLearn: a site that provides courses for C++, Python 3, Java, JavaScript, Ruby and several other programming languages.
  • hackr.io: a site that collects links to good programming courses and tutorials.
  • Code.org: an American non-profit organisation that aims to expand access to computer science in schools and to increase participation by women and underrepresented minorities. Its couse catalogue contains courses for the following target groups: grades K–5, grades 6–12 and “University+”. The “University+” category contains links to courses hosted elsewhere, for example on edX, Codeacademy, Udemy and W3Schools.
  • Functional Programming in Haskell: Supercharge Your Coding: online course by the University of Glasgow hosted on FutureLearn. The course is six weeks long and requires four hours of study per week. The next session starts on 2 April 2018.

Popularity of Programming Languages

  • TIOBE Software: TIOBE Index: indicator of the popularity of programming languages.
  • Martin, Bob C.: The Last Programming Language, Skills Matter, 13.11.2011: 60-minute video of a presentation that Bob C. Martin gave in London in July 2011. See also the comments the blog Lambda the Ultimate (14.07.2011).
  • Conway, Drew: Ranking the popularity of programming languages, Dataists, 09.12.2010.
  • O'Grady, Stephen: The RedMonk Programming Language Rankings: September 2012, RedMonk, 12.09.2012. According to this ranking, the ten most popular languages were JavaScript, Java, PHP, Python, Ruby, C#, C++, C, Objective-C and Shell.
  • O'Grady, Stephen: The RedMonk Programming Language Rankings: June 2017, RedMonk, 08.06.2017. According to this ranking, the ten most popular languages were JavaScript, Java, Python, PHP, C#, C++, CSS, Ruby, C and Objective-C.
  • Most Popular Coding Languages of 2013, CodeEval blog, 06.02.2013.
  • Garling, Caleb: Why Do Some Programming Languages Live and Others Die?, Wired, 08.06.2012.
  • Svensson, Peter: The "Life of Brian" definition of language choices, Undescript, 21.08.2007.
  • Garling, Caleb: The Secret of a Successful Programming Language? A Really Great Beard, Wired, 15.06.2012.
