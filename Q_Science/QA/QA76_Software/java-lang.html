Editors and IDEs

  • NetBeans IDE: a free and open source IDE. NetBeans was originally developed as an IDE for Java (including Swing) but now also supports a number of other languages such as PHP.
    Note: in 2016, NetBeans was accepted for incubation at Apache. See also Apache NetBeans.
  • JetBrains: IntelliJ IDEA: the community edition of this tools is avaialable under the Apache 2.0 licence (open source).
  • Oracle JDeveloper: a free proprietary IDE deeloped by Oracle.
  • BlueJ: a free and open source Java development environment developed mainly for educational purposes. The IDE is written in Java and available across platforms. There is also a textbook that relies on BlueJ: Objects First with Java: A Practical Introduction using BlueJ by David J. Barnes and Michael Kölling.
  • DrJava: a lightweight development environment for writing Java programs. It is designed primarily for students, providing an intuitive interface and the ability to interactively evaluate Java code.
  • Greenfoot: another (open source) educational programming environment for Java. There is also a textbook that relies on Greenfoot: Introduction to Programming with Greenfoot by Michael Kölling.
  • JSource, a light Java IDE (licence: GNU GPL version 2).
  • JCreator: commercial Java IDE that is available both as an LE edition and a “Pro Edition”.

There are many other editors and IDEs that, among other languages, support Java, For example Eclipse, jEdit, Atom, Notepad++, Geany, GNU Emacs, vi, vim, jGRASP (which focuses on software visualization), Kate and Sublime Text.

Java Libraries and Software

  • Java-Source.net: open source software in Java.
  • Awesome Java: A curated list of awesome Java frameworks, libraries and software, maintained as a community effort.
  • Joda-Time: a quality replacement for the Java date and time classes (Apache 2.0 licence).
  • JToaster: for notification messages in Java Swing applications (Apache 2.0 licence).
  • Qt Jambi: a Java binding of the cross-platform application framework Qt (Qt Jambi on Wikipedia).
  • JogAmp: this website is the home of several libraries for 3D graphics, multimedia and processing: JOGL, JOCL and JOAL.
