(Brian Goetz still answers questions about parallel processing on Stack Overflow.)

Test-Driven Development and Unit Testing

  • Langr, Jeff; Hunt, Thomas; Thomas, Dave: Pragmatic Unit Testing in Java 8 with JUnit. Pragmatic Bookshelf, 2015. (234 pages) ISBN 978-1-94122-259-1.
  • Langr, Jeff: Agile Java: Crafting Code with Test-Driven Development. Prentice Hall, 2005. (792 pages) ISBN 978-0-13-265035-9 (e-book).
    Agile Java teaches test-driven development (TDD<) based on Java 5 and JUnit 3. Both Java and JUnit have changed significantly since 2005. For example, JUnit no longer has a graphical user interface (the famous green bar) and Java has introduced new features such as pluggable annotations (in Java 6), which are used extensively in JUnit 5 and lambda expressions (in Java 8). In April 2017, Langr wrote that he was tempted to start working on an Agile Java rewrite.
  • Mellor, Alan: How to write a TDD Unit Test with Java, View from the Code Face, June 2017. (Includes a 20-minute YouTube video.)
  • Richardson, Alan: TDD - Test Driven Development - Java JUnit FizzBuzz, Evil Tester, 04.03.2018. (Includes a 13-minute YouTube video; see also source code on GitHub.)

Editors and IDEs

  • NetBeans IDE: a free and open source IDE. NetBeans was originally developed as an IDE for Java (including Swing) but now also supports a number of other languages such as PHP.
    Note: in 2016, NetBeans was accepted for incubation at Apache. See also Apache NetBeans.
  • JetBrains: IntelliJ IDEA: the community edition of this tools is avaialable under the Apache 2.0 licence (open source).
  • Oracle JDeveloper: a free proprietary IDE deeloped by Oracle.
  • BlueJ: a free and open source Java development environment developed mainly for educational purposes. The IDE is written in Java and available across platforms. There is also a textbook that relies on BlueJ: Objects First with Java: A Practical Introduction using BlueJ by David J. Barnes and Michael Kölling.
  • DrJava: a lightweight development environment for writing Java programs. It is designed primarily for students, providing an intuitive interface and the ability to interactively evaluate Java code.
  • Greenfoot: another (open source) educational programming environment for Java. There is also a textbook that relies on Greenfoot: Introduction to Programming with Greenfoot by Michael Kölling.
  • JSource, a light Java IDE (licence: GNU GPL version 2).
  • JCreator: commercial Java IDE that is available both as an LE edition and a “Pro Edition”.

There are many other editors and IDEs that, among other languages, support Java, For example Eclipse, jEdit, Atom, Notepad++, Geany, GNU Emacs, vi, vim, jGRASP (which focuses on software visualization), Kate and Sublime Text.

Java Libraries and Software

  • Java-Source.net: open source software in Java.
  • Awesome Java: A curated list of awesome Java frameworks, libraries and software, maintained as a community effort.
  • EclEmma: Java code coverage for Eclipse; based on the JaCoCo code coverage library.
  • JaCoCo Java Code Coverage Library.
  • Joda-Time: a quality replacement for the Java date and time classes (Apache 2.0 licence).
  • JToaster: for notification messages in Java Swing applications (Apache 2.0 licence).
  • Qt Jambi: a Java binding of the cross-platform application framework Qt (Qt Jambi on Wikipedia).
  • JogAmp: this website is the home of several libraries for 3D graphics, multimedia and processing: JOGL, JOCL and JOAL.
  • JIDE Common Layer: open-source foundation for JIDE's user interface products. The JIDE Common Layer source code is dual-licensed under the conditions of the GPL with classpath exception and a free commercial licence.
  • SWT: The Standard Widget Toolkit: an open source widget toolkit for Java designed to provide efficient, portable access to the user-interface facilities of the operating systems on which it is implemented. See also Standard Widget Toolkit on Wikipedia.
  • SWTBot: an open-source Java based UI/functional testing tool for testing SWT, Eclipse and GEF based applications.
  • LWJGL - Lightweight Java Game Library: a Java library that enables cross-platform access to popular native APIs useful in the development of graphics (OpenGL, Vulkan), audio (OpenAL) and parallel computing (OpenCL) applications.
  • install4j: a powerful multi-platform Java installer builder that generates native installers and application launchers for Java applications.

Libraries and Tools for Testing

  • JUnit Team: JUnit5 (licences: Apache License v2.0 for junit-platform-surefire-provider; Eclipse Public License v2.0 for all other modules).
  • JUnit Team: Junit4 (licence: Eclipse Public License 1.0). JUnit 4.12 API. Since version 4, JUnit no longer has a graphical test runner.
  • TestNG: a testing framework inspired from JUnit and NUnit but introducing some new functionalities that make it more powerful and easier to use. TestNG source code is available on GitHub (Apache 2.0 licence). An Eclipse plugin for TestNG is also available.
  • Infinitest: The Continuous Test Runner for the JVM. See Infinitest for Eclipse and Infinitest for IntelliJ. Infinitest source code is available on GitHub.
